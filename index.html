Remote setup

For phone remote control across devices, edit firebase-config.js and add your Firebase web app config. The app still works offline for local presenting without Firebase.

window.PRESENTATION_HUB_FIREBASE_CONFIG = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
